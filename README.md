## 🛠️ 2. Arquitectura Técnica y Tecnologías

El sistema sigue un patrón de diseño **Modelo-Vista-Controlador (MVC)** para garantizar la escalabilidad exigida en el pliego:

* **Backend (Controlador):** Python con el framework **Flask** para la gestión de rutas y lógica de negocio.
* **Base de Datos (Modelo):** **MySQL**, almacenamiento persistente del stock con lógica de protección ante inventario negativo.
* **Frontend (Vista):** HTML5 y **Bootstrap 5**, logrando una interfaz intuitiva, moderna y 100% *responsive* para su uso en dispositivos móviles dentro de la tienda.
